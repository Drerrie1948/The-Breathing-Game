# Play Store release checklist

- Wrap the verified web build with Capacitor Android.
- Replace the prototype package id if the final studio domain differs.
- Generate adaptive launcher icons and a portrait splash screen.
- Test touch, audio resume, vibration and offline startup on physical Android devices.
- Produce a signed Android App Bundle (`.aab`).
- Prepare store screenshots, feature graphic, description and support contact.
- Complete Data safety, content rating and target-audience declarations.
- Run a closed Play Console test before production release.
